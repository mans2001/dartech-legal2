# Dartech Legal v2

**Dartech Legal v2** is a fully bilingual (English/Arabic) legal page system for the Dartech AI-powered smart home assistant platform. This project is designed to be simple, mobile-friendly, and **production-ready** for embedding in mobile apps (WebView) or deploying to static hosting platforms like [Vercel](https://vercel.com/).

---

## 📌 Features

- Fully bilingual: English 🇺🇸 and Arabic 🇪🇬
- Instant language switching (no page reload)
- Right-to-left layout for Arabic
- Mobile-first, responsive design
- Google Play & App Store compliant legal text
- Sections included:
  - Terms & Conditions
  - Privacy Policy
  - Acceptable Use Policy
  - Disclaimer
  - Third-Party Services
- No frameworks, zero build steps (pure HTML, CSS, JS)
- Placeholder for contact channel (ready for future update)
- Last updated: August 31, 2026
